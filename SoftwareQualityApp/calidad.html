¿Qué es la Calidad del Software?

Inicio Calidad Código Programación Normas Modelos Estándares Evaluación Modelo Conclusiones

La calidad del software es el grado en que un producto de software satisface los requisitos explícitos e implícitos, cumple con las expectativas de los usuarios y se adapta a criterios técnicos y de negocio. Incluye atributos como:

  • Funcionalidad: que el software realice las funciones requeridas.
  • Fiabilidad: comportamiento estable y recuperable frente a fallos.
  • Usabilidad: facilidad de uso por parte del usuario final.
  • Eficiencia: uso óptimo de recursos (tiempo y memoria).
  • Mantenibilidad: facilidad para corregir, mejorar o adaptar el software.
  • Portabilidad: capacidad de trasladar el software a otros entornos.

La calidad se alcanza mediante buenas prácticas de desarrollo, pruebas sistemáticas, revisiones de código y adopción de normas y modelos de calidad.
