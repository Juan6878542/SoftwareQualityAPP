Ejemplos de estándares y buenas prácticas:

  • IEEE: Familias de estándares para pruebas, documentación y procesos.
  • ISO: Normas internacionales aplicables a calidad y seguridad.
  • OWASP: Guía de buenas prácticas para seguridad de aplicaciones web.
  • Guías de codificación: Conjuntos de reglas para mantener consistencia y calidad en el código.
  • Métricas de calidad: Defect Density, Test Coverage, Code Complexity, etc.
