Normas de Calidad de Software

Inicio Calidad Código Programación Normas Modelos Estándares Evaluación Modelo Conclusiones

Algunas normas relevantes:

  • ISO/IEC 25010: Modelo de calidad para sistemas y software que define atributos de calidad (funcionalidad, fiabilidad, usabilidad, eficiencia, mantenibilidad, portabilidad).
  • ISO/IEC 9126: Predecesora de la 25010, enfocada en la evaluación de la calidad del software.
  • IEEE 730: Estándar para planes de aseguramiento de la calidad del software.
  • ISO 9001: Requisitos para sistemas de gestión de calidad aplicables a procesos de desarrollo de software.
  • IEEE 829 (829-2008): Guía para documentación de pruebas y reportes.
