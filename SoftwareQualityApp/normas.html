Normas de Calidad de Software

Inicio Normas Modelos Estándares Evaluación Modelo
  • ISO/IEC 25010: Define atributos de calidad como funcionalidad, fiabilidad, usabilidad, eficiencia, mantenibilidad y portabilidad.
  • ISO/IEC 9126: Norma internacional para evaluación de calidad de software.
  • IEEE 730: Estándar para planes de aseguramiento de la calidad.
  • ISO 9001: Sistemas de gestión de calidad aplicables a procesos de software.
  • IEEE 829: Estándar para documentación de pruebas.
