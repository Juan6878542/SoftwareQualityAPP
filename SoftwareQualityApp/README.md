# Calidad del Software - Aplicativo Educativo

Sitio estático con 10 páginas que explican conceptos de calidad del software, normas, modelos, estándares y un evaluador que calcula una calificación entre 0 y 5.

Archivos incluidos:
- index.html
- calidad.html
- codigo.html
- programacion.html
- normas.html
- modelos.html
- estandares.html
- evaluacion.html
- modelo.html
- conclusiones.html
- styles.css
- app.js

Cómo usar:
1. Descomprimir y abrir `index.html` en un navegador, o subir al repositorio GitHub y activar GitHub Pages, o importar a Replit (HTML/CSS/JS).
2. En la página Evaluación, complete los criterios y presione "Calcular" para ver la puntuación final; también puede exportar el resultado en JSON.
