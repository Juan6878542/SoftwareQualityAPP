Modelo de Evaluación Cuantitativa

Inicio Calidad Código Programación Normas Modelos Estándares Evaluación Modelo Conclusiones

El modelo calcula promedios por categoría y luego una puntuación final como media ponderada:

  • Estándares de Calidad (peso 50%)
  • Código (peso 30%)
  • Programación (peso 20%)

Cada criterio se ingresa de 0 a 5. El resultado es una puntuación numérica entre 0 y 5 y un desglose porcentual. Además, el resultado puede exportarse en JSON para registro.
