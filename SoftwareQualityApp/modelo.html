Modelo de Evaluación Cuantitativa

Inicio Normas Modelos Estándares Evaluación Modelo

El modelo calcula promedios por categoría y luego una puntuación final como media ponderada:

  • Estándares de Calidad (peso 50%)
  • Código (peso 30%)
  • Programación (peso 20%)

Cada criterio se ingresa de 0 a 5. El resultado es una puntuación numérica entre 0 y 5 y un desglose porcentual.
